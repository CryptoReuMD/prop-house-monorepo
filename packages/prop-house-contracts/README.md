# Prop House Contracts

## Development

### Install dependencies

### Python

```sh
python3.7 -m venv ~/cairo_venv
source ~/cairo_venv/bin/activate
pip3 install -r requirements.txt
```

### Submodules

```sh
make install
```

### Run tests

```sh
make test
```

### Lint Contracts

```sh
make lint
```

### Build Contracts

```sh
make build
```
